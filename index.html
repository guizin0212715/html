Passos para Hospedar um Projeto no GitHub Pages

  1. Crie um repositório no GitHub.
  2. no terminal incialize o repositorio local com git init.
  3. Altere o nome da branch principal para main
  4. adicione os arquivos com git add e faça o commit com git commit -m "first commit"
  5. Envie as alterações para o GitHub.
  6. Vá até as configurações do repositório no GitHub.
  7. Ative o GitHub Pages na aba "Pages".
  8. Aguarde alguns minutos e acesse o link fornecido.
  9. Apos alguns segundos, seu site estará disponivel no link exibido em "Pages"
